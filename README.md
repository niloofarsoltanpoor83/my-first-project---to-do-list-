# 📝 To-Do List App

A modern To-Do List application built with HTML, CSS and JavaScript.

## ✨ Features
- Add tasks
- Edit tasks
- Delete tasks
- Search tasks
- Sort by date and priority
- Due dates
- Overdue warning
- Dark mode
- Progress bar
- Local Storage

## 🛠 Technologies
- HTML5
- CSS3
- JavaScript
